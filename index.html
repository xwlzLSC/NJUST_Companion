备用导入方案
书签脚本导入
如果本地直连登录不可用，仍可通过浏览器打开教务系统页面后运行书签脚本导入。每次同步只更新当前模块，其他数据会自动保留。
1. 进入教务系统并完成登录。
2. 打开“课表 / 成绩 / 考试安排”其中一个页面。
3. 点击浏览器书签中的脚本，复制生成的 JSON。
4. 回到本应用，在“粘贴导入”中确认同步。

关于
南理教务助手
一个本地优先、可安装、可离线查看，并支持本机代理直连教务系统的南京理工大学教务数据平替方案。
为了避免把账号密码交给第三方服务器，浏览器版的直连登录由你自己的电脑本地 Node 服务完成；安卓安装包则改为手机本机原生联网同步，前端只负责展示与缓存。
导入同步数据
支持两种内容：1）书签脚本从教务页面复制的单模块 JSON；2）本应用导出的整份备份。导入时会自动合并，不会把其他模块清空。
成绩帮助
1. 英语课默认不勾选：多数学院英语课程更偏向奖学金统计，保研更常看四六级最高分；所以初始会优先勾选非英语必修课，你也可以手动改。

2. 四六级计入规则：默认取 CET4 / CET6 历次中的最高总分，按 `总分 ÷ 710 × 100` 换算成百分制，再按 `8 学分` 计入“已选+四六级”。

3. 文本成绩换算：及格=60，中等=70，良好=80，优秀=90；同时兼容 `优- / 良+ / 良` 这类成绩。

4. 绩点公式：`<60=0.0`，`<64=1.0`，`<68=1.5`，`<72=2.0`，`<75=2.3`，`<78=2.7`，`<82=3.0`，`<85=3.3`，`<90=3.7`，`>=90=4.0`。
