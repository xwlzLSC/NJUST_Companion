# 南理教务助手 (NJUST Companion)

一个专为在校学生量身打造的**跨平台、本地优先**新一代全栈教务系统客户端。本项目旨在通过极为优雅的原生级视觉设计，配合底层黑魔法级别的防掉线抓取架构，提供一触即达的极致体验。

---

## 🌟 核心杀手级亮点

### 1. 极致的原生级果味 UI
- **微卡片美学**：彻底摒弃传统系统枯燥厚重的排版，大量采用 $56\times56$ 微徽标、高斯模糊、弥散阴影和呼吸态间距，数据呈现如 iOS 般轻盈灵动。
- **全面屏沉浸**：加持 `viewport-fit=cover` 配合底部与顶部安全区补偿计算，在任何手机上均能无缝避避刘海与底部横条。
- **直觉交互**：周视图 / 学期视图自由滑切，平滑滚动，交互质感拉满。

### 2. “免妥协”：智能双模鉴权安全网
为彻底解决验证码干扰与频繁掉线，首创这套由双极化机制组成的最强登录底座：
- **前台手动守底**：UI 提供明文 **验证码（选填）** 框，完美照顾缺少 WebAssembly 算力的任何古董机型，保证 100% 能登入。
- **后台暗区 OCR 连载**：搭载底层精简版 `Tesseract.js`。若您长达半小时未开启 App 导致服务器强制拉黑凭证——在您切回应用的一瞬间，系统会召唤后台 OCR 进行最高 $15$ 次不间断的无头图片获取与盲猜，为您无感重登，把所有的恶心报错弹窗杀在摇篮里！

### 3. 跨端 PWA 部署，斩断苹果上架枷锁
- **iOS 苹果免签零负担**：无需 Mac 电脑，告别 Xcode！通过浏览器内置底层的 PWA 机制。在同一网络下单点 Safari 【添加到主屏幕】，桌面立刻生成零边界全屏的独立应用！不受查杀，永远在线。
- **双端原生支持**：依然保留 Capacitor 对 Android 的强力原生打包能力 (`.apk` 裸包)，以及 Node 桌面端的网络代理守护。

### 4. 灭霸级 Cookie 快照
- 针对手机切后台疯狂回收 WebKit 短效内存的系统尿性，自研 `tryRecoverSession` 强注流派：不管您杀后台杀得多干净，只要您刚切回前台网络受阻，系统直接调取我们在本地硬盘给您私藏的【无损纯净 Cookie 快照】进行暴力覆盖注射，强行续命，让你的课表永不掉线。

## 🛠️ 主要功能速览

- **首页聚合**：当前时间线胶囊、快捷大入口。
- **课表与日历**：按真实教学楼编排，精算周次并适配各种“大课/连堂”。
- **强力成绩单**：一键计算加权平均分、GPA 分段匹配，自动熔入四六级大统分。
- **深海考试雷达**：考场定位计算与倒计时。
- **空教室追踪**：连包调取，精准按校区与教学大楼扫描静谧自习室。

## 🚀 起步与运行方式

### 环境基础：Node + 浏览器模式
```bash
# 1. 挂载所有依赖引擎
npm install
# 2. 启动强力本地代理服（负责反制教务处反爬策略并代理跨域）
npm start
```
打开 `http://127.0.0.1:3030` 即可享受顺滑端体验。

### 苹果 iOS 部署（局域网/服务器挂网 PWA 方式）
1. 在同一内网（或您的私有云）下启动 `npm start`。
2. 在 iPhone **Safari** 中输入对应网址。
3. 点击 Safari 底部中间的**分享**按钮，点击**添加到主屏幕**即可！

### 安卓原生 APK 编译
保证配置好 JDK 21 与 Android SDK。
```bash
npm run apk:debug
```
最终产出位于：`android/app/build/outputs/apk/debug/app-debug.apk`

## 🧩 极客向：抓取机制与底层模型

教务系统不是 JSON API 的乐园，而是一个混合了 UTF-8 与 GBK 历史债务的传统 Java JSP Web。我们的核心解析器 `parser.js` 做到了极强的脏数据抹平：
- **精准投递**：在 `server.js` 和 `native-sync.js` 对请求实施拦截器包装，解决反手给你抛出一个包含 `*必填` 恶意弹窗的异常拦截。
- **智能重抽**：不仅应对新版教务（如 `NEW_XSD_PYGL`），并且针对 DOM 层深层抓取封装了 `createDom` 洗地器。
- **隔离安全**：手机上的任何脱机操作完全落入本地的 `IndexedDB + LocalStorage` 数据隔离池。绝无任何密码、票据向第三方云平台泄露的可能性。
